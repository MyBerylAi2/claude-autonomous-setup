## Slash Commands

| Command | Action |
|---------|--------|
| `/spec <topic>` | Create MarkItDown spec (no code) |
| `/build` | Begin implementation after spec approval |
| `/state` | Save session with AMANDA-ID |
| `/resume` | Continue from saved state |
| `/skills` | Reload all skills |
| `/autonomous` | Reinforce autonomous mode |

## Session Persistence

Every session gets an AMANDA-ID:
```
AMANDA-20260120-0215-7X9K
```

This enables:
- Auto-versioned backups
- Resume after disconnect
- Session tracking across projects
